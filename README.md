## Overview
AI Study Companion helps students learn any topic efficiently using Generative AI. Users can get:
- Simple explanations
- Revision notes
- Practice questions

This project demonstrates **Python scripting, prompt engineering, and API integration**, with both **terminal-based** and **web-based (Streamlit) versions**.

---

## Features
- Multi-feature menu (Explanation / Notes / Questions)  
- Save AI output for future reference  
- Beginner-friendly interface  
- Modular prompt design  
- Streamlit GUI for web use  

---

## Tech Stack
- Python  
- Groq API (Generative AI)  
- Streamlit (GUI)  
- Prompt Engineering
